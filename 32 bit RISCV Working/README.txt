To run this on pc.

Step1- Download Icarus Verilog:  https://sourceforge.net/projects/iverilog/

  Download the appropriate installer for your operating system.
  Install Icarus Verilog:

Step2- Install GTKWave

  Download GTKWave:  https://gtkwave.sourceforge.net/
  Download the appropriate installer for your operating system.

Step3- Copy this repo in a folder

Step4- Open the folder in the terminal

Step5- Make the script executable by running the following command in your terminal:
            chmod +x run.sh
Step6- Execute the script by running:
          ./run.sh
    This script will compile the Verilog testbench, run the simulation, and open      the waveform in GTKWave. If any step fails, it will     print an error message and exit
